Travel Planner
  • Clone or download the project on local machine
  • Open terminal and navigate to the 'travel planner' folder with the commmand 'cd'
    (ex: 'cd Desktop' to enter the 'Desktop' directory, and 'cd travel-planner' to access the travel-planner folder)
  • Run the command 'npm install' to install the necessary packages
  • Run the command 'npm run start' to start the server

Desktop or laptop only
